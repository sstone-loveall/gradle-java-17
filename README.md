# gradle-java-17
Application template for Java 17 applications with Gradle package management.

## Project Requirements
- Gradle 8.x
- JDK 17

## Download Dependencies
### Clean install
```shell
make clean
make deps
```

### Refresh dependencies
```shell
make deps
```



## Build
```shell
make build
```

## Test
```shell
make test
```

## Run
## Test
```shell
make run
```
